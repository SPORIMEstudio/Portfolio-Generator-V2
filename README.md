
# 🚀 Portfolio Generator V2

**Portfolio Generator V2** is a simple and powerful tool that automatically generates a clean and modern developer portfolio. It helps developers quickly create a professional portfolio website without spending hours designing or coding it manually.

---

## ✨ Features

- ⚡ **Fast Setup** – Create a portfolio in seconds.
- 🎨 **Clean UI** – Modern and minimal portfolio design.
- 📱 **Responsive Layout** – Works on desktop and mobile devices.
- 🧩 **Easy Customization** – Modify sections easily.
- 📄 **Export Ready** – Generated portfolio can be exported and deployed anywhere.

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

---


---

## ⚙️ How to Use

1. Clone the repository

```bash
git clone https://github.com/SPORIMEstudio/Portfolio-Generator-V2.git
```
2. Open the project folder
```bash
cd Portfolio-Generator-V2
```

Run the project

Simply open index.html in your browser.
